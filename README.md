Descrição

O Sistema para Imobiliária é uma aplicação desenvolvida com os conceitos de Programação Orientada a Objetos (POO) que tem como objetivo facilitar o gerenciamento de imóveis, clientes, corretores, visitas e propostas de compra ou aluguel.

O sistema busca centralizar as informações da imobiliária em uma única plataforma, proporcionando maior organização, controle e eficiência nos processos de negociação e administração dos imóveis.

Principais Funcionalidades
Cadastro e gerenciamento de imóveis;
Cadastro de clientes e corretores;
Registro de visitas aos imóveis;
Envio e acompanhamento de propostas;
Consulta e filtragem de imóveis;
Controle do status dos imóveis (disponível, alugado ou vendido).
Tecnologias Utilizadas
C#
Programação Orientada a Objetos (POO)
JSON para armazenamento de dados
Git e GitHub para versionamento
Conceitos de POO Aplicados
Abstração: modelagem das entidades do sistema.
Encapsulamento: proteção e controle de acesso aos dados.
Herança: reutilização de características comuns entre classes.
Polimorfismo: comportamentos específicos para diferentes tipos de objetos.
Objetivo do Projeto

Desenvolver uma solução prática para o gerenciamento de uma imobiliária, aplicando os princípios da Programação Orientada a Objetos e boas práticas de desenvolvimento de software.
